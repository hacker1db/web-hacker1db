```javascript

    console.log(`Hello, ${name}!`);
```

## Python Example

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# Generate fibonacci sequence
for i in range(10):
    print(f"F({i}) = {fibonacci(i)}")
```
